# Bike Sales Data Dashboard

Project Overview
This project focuses on analyzing bike sales data and creating an interactive **Power BI dashboard** to derive meaningful business insights. The project demonstrates the complete data analysis workflow starting from raw data to a final visual dashboard.


Objectives
- Understand bike sales trends across different customer segments
- Perform data cleaning and normalization
- Create an interactive dashboard for decision-making
- Present insights using visual analytics

Tools & Technologies Used
Microsoft Excel– Data cleaning and normalization
Power BI – Data modeling and dashboard creation
GitHub – Version control and project documentation
